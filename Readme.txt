AddressBook
--------------------------------------
Простенькая адресная книга, работающая с БД MySQL.

Все операции проходят через транзакции,
используются внутренние и внешние ключи для
соединения таблиц.

Поиск пока работает по фамилии, с учетом того,
что каждая фамилия в БД уникальна.

Приложен файл с инструкциями sql для проверки -
/resources/addressbook.sql


QueryViewer
--------------------------------------
С помощью этой программы можно запускать
произвольные запросы, и увидеть в таблице результат.

Если поменяете в файле с настройками название БД
на свою, сможете увидеть свою базу.
